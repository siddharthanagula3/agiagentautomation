## Pre-Publication Checklist

### SEO Elements

- [ ] **H1 Title**
  - [ ] 40-60 characters total
  - [ ] Primary keyword in first 2-3 words
  - [ ] Includes specific angle or benefit
  - [ ] No keyword stuffing or hype

- [ ] **Meta Description**
  - [ ] 150-160 characters
  - [ ] Includes primary keyword naturally
  - [ ] Contains hook + benefit + CTA
  - [ ] Compelling reason to click

- [ ] **Keyword Strategy**
  - [ ] Primary keyword in first 100 words
  - [ ] Keyword appears 1-2% of total words (not stuffed)
  - [ ] Secondary keywords in H3s and body
  - [ ] All keywords used naturally in context

- [ ] **Internal Links**
  - [ ] 3-5 links minimum
  - [ ] Links are relevant to topic
  - [ ] Anchor text describes destination
  - [ ] Mix of blog posts + product pages

- [ ] **Headers**
  - [ ] Exactly one H1
  - [ ] 4-6 H2s organizing major sections
  - [ ] H3s under each H2 for subsections
  - [ ] No skipped header levels

### Content Quality

- [ ] **Word Count:** 1200-2000 words
- [ ] **Data/Stats:** All claims backed by data or sources
- [ ] **Examples:** 2-3 specific real-world examples
- [ ] **Callout Boxes:** 1-2 highlighted insights
- [ ] **Readability:** Short paragraphs, active voice, clear topic sentences

### Structure

- [ ] **Opening Hook:** Provocative, relevant, creates curiosity
- [ ] **Context:** Explains why this matters now
- [ ] **Core Content:** 2-3 main sections with specific examples
- [ ] **"What It Means For You":** 2-3 reader personas with specific implications
- [ ] **"Looking Ahead to 2026":** 3 specific predictions with justifications
- [ ] **Key Takeaways:** 3-4 distinct insights in actionable bullets
- [ ] **CTA:** 2-3 action-oriented next steps

### Tone & Voice

- [ ] **Conversational:** Reads naturally, not overly formal
- [ ] **Authoritative:** Specific claims, data-backed, confident
- [ ] **No Jargon:** Avoids corporate speak, explains concepts clearly
- [ ] **Reader-Centric:** Focuses on reader benefit, not self-promotion
- [ ] **No Hype:** Practical implications, not exaggeration

## Publishing Workflow

1. **Planning** (30 min)
   - [ ] Choose topic and primary keyword
   - [ ] Identify reader personas
   - [ ] Outline 3-4 main sections
   - [ ] Identify data sources

2. **Writing** (2-3 hours)
   - [ ] Write draft following template
   - [ ] Include specific examples and data
   - [ ] Add internal links (3-5 minimum)

3. **First Review** (30 min)
   - [ ] Check structure and flow
   - [ ] Verify all claims have sources
   - [ ] Improve clarity and readability
   - [ ] Add callout boxes for emphasis

4. **SEO Optimization** (30 min)
   - [ ] Refine H1 title (40-60 chars, keyword first)
   - [ ] Write meta description (150-160 chars)
   - [ ] Check keyword placement (first 100 words, 1-2% density)
   - [ ] Verify header hierarchy
   - [ ] Review internal links

5. **Final Review** (15 min)
   - [ ] Proofread for typos and grammar
   - [ ] Check formatting consistency
   - [ ] Verify all links work
   - [ ] Confirm reading time estimate

6. **Publishing**
   - [ ] Add metadata (author, date, tags, featured image)
   - [ ] Schedule social media promotion
   - [ ] Share in company newsletter
   - [ ] Add to blog index
