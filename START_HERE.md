## 👉 STEP-BY-STEP (Path C)

### Step 1: Open Another Browser Tab
**Right now**, open: https://supabase.com

Keep this document open in one tab, Supabase in another.

### Step 2: Create Supabase Account
1. Click "Start your project"
2. Sign in with GitHub (recommended) or email
3. Wait for dashboard to load

### Step 3: Create New Project
1. Click "New Project"
2. Fill in:
   - **Name**: agiagent (or any name)
   - **Database Password**: Create strong password (save it!)
   - **Region**: Choose closest to you
   - **Pricing Plan**: Free
3. Click "Create new project"
4. ⏰ Wait 2-3 minutes for setup

### Step 4: Get Your Credentials
Once project is ready:

1. Click "Settings" (gear icon in sidebar)
2. Click "API"
3. You'll see:
   - **URL**: Copy this (looks like https://xxxxx.supabase.co)
   - **anon public**: Copy this (long string starting with eyJ...)

### Step 5: Update .env File
1. Open your project folder
2. Find file: `.env`
3. Open it in text editor
4. Replace these lines:
   ```
   VITE_SUPABASE_URL=https://xxxxx.supabase.co
   VITE_SUPABASE_ANON_KEY=eyJxxx...xxx
   ```
   With your REAL values from Supabase

5. Save the file

### Step 6: Get AI API Key
Choose ONE (Anthropic recommended):

**Option A: Anthropic Claude**
1. Go to: https://console.anthropic.com
2. Sign up/Sign in
3. Click "API Keys"
4. Click "Create Key"
5. Copy the key (starts with sk-ant-)
6. In .env file, replace:
   ```
   VITE_ANTHROPIC_API_KEY=sk-ant-your-real-key-here
   ```

**Option B: Google Gemini**
1. Go to: https://makersuite.google.com/app/apikey
2. Sign in with Google
3. Click "Create API Key"
4. Copy the key (starts with AIza)
5. In .env file, replace:
   ```
   VITE_GOOGLE_AI_API_KEY=AIza-your-real-key-here
   ```

### Step 7: Verify Configuration
Open terminal/command prompt in your project folder and run:

```bash
node preflight-check.js
```

**Expected Output**:
```
✅ Node.js version
✅ package.json
✅ Dependencies installed
✅ .env file
✅ Supabase configured
✅ AI API configured
✅ Project structure
✅ Migration files
✅ TypeScript config

PRE-FLIGHT CHECK PASSED!
```

If you see ❌ for anything, fix it before continuing.

### Step 8: Setup Database
1. Go back to Supabase dashboard
2. Click "SQL Editor" in left sidebar
3. Click "New Query"
4. Open file: `supabase/migrations/001_initial_schema.sql`
5. Copy ALL contents (Ctrl+A, Ctrl+C)
6. Paste into Supabase SQL Editor (Ctrl+V)
7. Click "Run" (or Ctrl+Enter)
8. Wait for "Success"

**Repeat for all 5 migrations**:
- 001_initial_schema.sql ✅
- 003_settings_tables.sql
- 004_complete_workforce_schema.sql
- 005_analytics_tables.sql
- 006_automation_tables.sql

### Step 9: Verify Database
In Supabase SQL Editor, run:

```sql
SELECT COUNT(*) as total_tables
FROM information_schema.tables 
WHERE table_schema = 'public';
```

**Expected**: At least 18 tables

If you get this, YOU'RE READY! 🎉

### Step 10: Start Application
In your terminal:

```bash
npm run dev
```

Wait for:
```
Local: http://localhost:5173
```

Open that URL in your browser!

### Step 11: Test It
1. Click "Sign Up"
2. Enter email and password
3. Create account
4. Should redirect to dashboard
5. Click "Chat"
6. Send a message
7. Get AI response

**If everything works**: CONGRATULATIONS! 🎉

### Step 12: Build for Production
